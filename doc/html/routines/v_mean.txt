V_MEAN


 PURPOSE:
       Computes the mean values of the given array of column vectors
	in the x direction.

 CATEGORY:
       UTIL/V


 CALLING SEQUENCE:
       result = v_mean(v)


 ARGUMENTS:
  INPUT:
       v:     An array of nv x nt column vectors (i.e. nv x 3 x nt).


  OUTPUT:
       NONE

 RETURN:
       Array of 1 x nt column vectors (i.e., 1 x 3 x nt).


 STATUS:
       Completed.


 MODIFICATION HISTORY:
       Written by:     Spitale
