GET_CORE_USER


 PURPOSE:
       Returns the user associated with an object descriptor


 CATEGORY:
       NV/LIB/TOOLS


 CALLING SEQUENCE:
       result = get_core_user(ods)


 ARGUMENTS:
  INPUT:
            ods:       An array of Object descriptors

  OUTPUT:
       NONE

 RETURN:
       An array of users.

 STATUS:
       Completed.
