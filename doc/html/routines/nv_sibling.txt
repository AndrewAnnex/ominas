NV_SIBLING


 PURPOSE:
	Returns the sibling data descriptor associated with a data 
	descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	sibling_dd = nv_sibling(dd)


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: 
	Data descriptor of the sibling.


 STATUS:
	Complete
