SLD_SET_OPACITY


 PURPOSE:
	Replaces the opacity of each given solid descriptor.


 CATEGORY:
	NV/LIB/SLD


 CALLING SEQUENCE:
	sld_set_opacity, slx, opacity


 ARGUMENTS:
  INPUT: 
	slx:	 Any subclass of SOLID.

	opacity:	 New opacity value.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
