MAP_FN_IMAGE_TO_MAP


 PURPOSE:
	Returns the name of the image->map function for each given map 
	descriptor.


 CATEGORY:
	NV/LIB/MAP


 CALLING SEQUENCE:
	fn_image_to_map = map_fn_image_to_map(md)


 ARGUMENTS:
  INPUT: NONE
	md:	 Array (nt) of map descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 

  OUTPUT: NONE


 RETURN:
	Array (nt) of image->map function names associated with each given 
	map descriptor.


 STATUS:
	Complete
