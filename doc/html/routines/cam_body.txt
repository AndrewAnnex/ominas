CAM_BODY


 PURPOSE:
	Returns the body descriptor for each given camera descriptor.


 CATEGORY:
	NV/LIB/BOD


 CALLING SEQUENCE:
	bd = cam_body(cd)


 ARGUMENTS:
  INPUT: 
	cd:	 Array (nt) of CAMERA descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	Body descriptor associated with each given camera descriptor.


 STATUS:
	Complete
