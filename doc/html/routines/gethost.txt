GETHOST


 PURPOSE:
	Determines the host machine type.


 CATEGORY:
	UTIL


 CALLING SEQUENCE:
	result=gethost()


 ARGUMENTS:
  INPUT: NONE

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT:
	status:	If no errors occur, status will be zero, otherwise
		it will be a string giving an error message.


 RETURN:
	String indicating the host machine type.


 STATUS:
	Complete
