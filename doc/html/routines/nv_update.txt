NV_UPDATE


 PURPOSE:
	Returns the update flag associated with a data descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	update = nv_update(dd)


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: 
	Data descriptor update flag.


 STATUS:
	Complete
