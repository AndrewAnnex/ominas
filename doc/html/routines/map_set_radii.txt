MAP_SET_RADII


 PURPOSE:
	Replaces the reference radii for each given map descriptor.


 CATEGORY:
	NV/LIB/MAP


 CALLING SEQUENCE:
	map_set_radii, md, radii


 ARGUMENTS:
  INPUT: 
	md:	 Array (nt) of map descriptors.

	radii:	 Array (3,nt) of new reference radii.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete

 	Written by:	Spitale, 1/1998
