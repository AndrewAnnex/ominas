PGS_NPOINTS


 PURPOSE:
	Counts points in an array of points structures.


 CATEGORY:
	NV/PGS


 CALLING SEQUENCE:
	n = pgs_npoints(pp)


 ARGUMENTS:
  INPUT: 
	pp:	Array of points structures

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 
	visible:	If set, only visible points are counted.

  OUTPUT: NONE


 RETURN: 
	Total number of points in the array of points structures.
