GLB_J


 PURPOSE:
       Returns the zonal harmonics for each given globe descriptor.


 CATEGORY:
	NV/LIB/GLB


 CALLING SEQUENCE:
	mass = glb_j(gbx)


 ARGUMENTS:
  INPUT:
	gbx:	 Array (nt) of any subclass of GLOBE descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:  NONE

  OUTPUT: NONE


 RETURN:
       Array (nj,nt) of zonal harmonics associated with each given globe 
	descriptor.


 STATUS:
	Complete
