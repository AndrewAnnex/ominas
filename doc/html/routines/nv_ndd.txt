NV_NDD


 PURPOSE:
	Returns the global maintenance ndd value.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	data = nv_ndd(dd)


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	The global maintenance ndd value.


 STATUS:
	Complete
