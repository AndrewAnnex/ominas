CAS_NAC_BOX


 PURPOSE:
	This procedure converts Cassini ISS NAC image corners into
	ISS WAC pixel locations.  Useful for plotting NAC FOV in WAC
	images.


 CATEGORY:
       UTIL/CASSINI


 CALLING SEQUENCE:
       result = cas_nac_box()


 ARGUMENTS:
  INPUT:
       NONE

  OUTPUT:
       NONE 

 KEYWORDS:

       NONE

 RETURN:
        Array of corner pixels.


 STATUS:
       Completed.


 MODIFICATION HISTORY:
       Written by:     Haemmerle 1/2000
