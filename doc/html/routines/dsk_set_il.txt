DSK_SET_IL


 PURPOSE:
	Replaces il in each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	dsk_set_il, bx, il


 ARGUMENTS:
  INPUT: 
	dkx:	 Array (nt) of any subclass of DISK.

	il:	 New il value.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
