DSK_ECC


 PURPOSE:
	Returns ecc for each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	ecc = dsk_ecc(dkx)


 ARGUMENTS:
  INPUT: NONE
	dkx:	 Any subclass of DISK.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 

  OUTPUT: NONE


 RETURN:
	ecc value associated with each given disk descriptor.


 STATUS:
	Complete
