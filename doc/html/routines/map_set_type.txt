MAP_SET_TYPE


 PURPOSE:
	Replaces the type name for each given map descriptor.


 CATEGORY:
	NV/LIB/MAP


 CALLING SEQUENCE:
	map_set_type, md, type


 ARGUMENTS:
  INPUT: 
	md:	 Array (nt) of map descriptors.

	type:	 Array (nt) of new type names.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
