GLB_PHASE_FN


 PURPOSE:
       Returns the name of the phase function for each given globe 
	descriptor.


 CATEGORY:
	NV/LIB/GLB


 CALLING SEQUENCE:
	mass = glb_phase_fn(gbx)


 ARGUMENTS:
  INPUT:
	gbx:	 Array (nt) of any subclass of GLOBE descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:  NONE

  OUTPUT: NONE


 RETURN:
       Array (nt) of phase function names associated with each given globe 
	descriptor.


 STATUS:
	Complete
