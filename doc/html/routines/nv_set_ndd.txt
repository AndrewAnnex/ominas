NV_SET_NDD


 PURPOSE:
	Sets a new ndd value in the NV state structure.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	nv_set_ndd, ndd


 ARGUMENTS:
  INPUT:
	ndd:	New ndd value.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
