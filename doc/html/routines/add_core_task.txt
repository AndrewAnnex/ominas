ADD_CORE_TASK

 CALLING SEQUENCE:
       add_core_task, ods, task


 ARGUMENTS:
  INPUT:
        ods:   Object descriptor

       task:   Name of task

  OUTPUT:
       NONE

 STATUS:
       Completed.
