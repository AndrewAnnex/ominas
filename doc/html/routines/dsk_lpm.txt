DSK_LPM


 PURPOSE:
	Returns lpm for each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	lpm = dsk_lpm(dkx)


 ARGUMENTS:
  INPUT: NONE
	dkx:	 Any subclass of DISK.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 

  OUTPUT: NONE


 RETURN:
	lpm value associated with each given disk descriptor.


 STATUS:
	Complete
