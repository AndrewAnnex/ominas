SLD_GM


 PURPOSE:
       Returns the GM value for each given solid descriptor.


 CATEGORY:
	NV/LIB/SLD


 CALLING SEQUENCE:
	gm = sld_gm(slx)


 ARGUMENTS:
  INPUT:
	slx:	 Array (nt) of any subclass of SOLID descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:  NONE

  OUTPUT: NONE


 RETURN:
       Array (nt) of GM values associated with each given solid descriptor.


 STATUS:
	Complete
