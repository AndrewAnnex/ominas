NV_DEREFERENCE


 PURPOSE:
	Turns an array of pointers to descriptors into an array of descriptors.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	result = nv_dereference(dp)


 ARGUMENTS:
  INPUT:
	dp:	Array of pointers to an arbitrary type of descriptor.

  OUTPUT:
	NONE


 KEYWORDS: NONE


 RETURN:
	Array of descriptors (structures).


 STATUS:
	Complete


 SEE ALSO:
	nv_rereference
