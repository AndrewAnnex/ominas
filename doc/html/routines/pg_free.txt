PG_FREE


 PURPOSE:
	Frees all pointers in the given structure.


 CATEGORY:
	NV/PG


 CALLING SEQUENCE:
	pg_free, s


 ARGUMENTS:
  INPUT:
	s:	Array of structures containing pointers to be freed.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 PROCEDURE:
	All fields of the structure are searched for pointers to free.


 STATUS:
	Complete
