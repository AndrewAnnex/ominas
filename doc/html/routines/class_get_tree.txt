CLASS_GET_TREE


 PURPOSE:
	Builds a class tree for te givern descriptor.


 CATEGORY:
	NV/LIB/CLASS


 CALLING SEQUENCE:
	classes = class_get_tree(od)


 ARGUMENTS:
  INPUT:
	od:	 Descriptor of any class.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:  NONE

  OUTPUT: NONE


 RETURN:
	String array giving the names of all classes in od, in descending order.


 STATUS:
	Complete
