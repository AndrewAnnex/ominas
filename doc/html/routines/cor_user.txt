COR_USER


 PURPOSE:
	Returns the username for each given core descriptor.


 CATEGORY:
	NV/LIB/COR


 CALLING SEQUENCE:
	user = cor_user(crx)


 ARGUMENTS:
  INPUT: NONE
	crx:	 Any subclass of CORE.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 

  OUTPUT: NONE


 RETURN:
	Username associated with each given core descriptor.


 STATUS:
	Complete
