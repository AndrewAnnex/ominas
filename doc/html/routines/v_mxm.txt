V_MXM


 PURPOSE:
       Performs matrix multiplication.

 CATEGORY:
       UTIL/V


 CALLING SEQUENCE:
       result = v_mxm(m1, m2)


 ARGUMENTS:
  INPUT:
       m1:     An array of nt matrices (i.e. 3 x 3 x nt).

       m1:     Another array of nt matrices.


  OUTPUT:
       NONE

 RETURN:
       Array of nt matrix products (i.e. 3 x 3 x nt).


 STATUS:
       Completed.
