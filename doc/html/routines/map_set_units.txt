MAP_SET_UNITS


 PURPOSE:
	Replaces the units for each given map descriptor.


 CATEGORY:
	NV/LIB/MAP


 CALLING SEQUENCE:
	map_set_units, md, units


 ARGUMENTS:
  INPUT: 
	md:	 Array (nt) of map descriptors.

	units:	 Array (2,nt) of new map units.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete

 	Written by:	Spitale, 1/1998
