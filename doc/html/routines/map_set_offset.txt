MAP_SET_OFFSET


 PURPOSE:
	Replaces the offset for each given map descriptor.


 CATEGORY:
	NV/LIB/MAP


 CALLING SEQUENCE:
	map_set_offset, md, offset


 ARGUMENTS:
  INPUT: 
	md:	 Array (nt) of map descriptors.

	offset:	 Array (2,nt) of new map offsets.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
