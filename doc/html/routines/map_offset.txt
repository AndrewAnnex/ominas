MAP_OFFSET


 PURPOSE:
	Returns the offset for each given map descriptor.


 CATEGORY:
	NV/LIB/MAP


 CALLING SEQUENCE:
	offset = map_offset(md)


 ARGUMENTS:
  INPUT: NONE
	md:	 Array (nt) of map descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 

  OUTPUT: NONE


 RETURN:
	Array (2,nt) of offsets associated with each given map descriptor.


 STATUS:
	Complete
