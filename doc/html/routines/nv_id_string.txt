NV_ID_STRING


 PURPOSE:
	Returns the identification string associated with a data descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	data = nv_id_string(dd)


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	The identification string associated with the data descriptor.


 STATUS:
	Complete
