NV_SET_UPDATE


 PURPOSE:
	Changes the update flag in a data descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	nv_set_update, dd, update


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor.

	update:	New update flag.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
