CAM_SET_ORIGIN


 PURPOSE:
       Replaces the origin array for each given camera descriptor.


 CATEGORY:
	NV/LIB/CAM


 CALLING SEQUENCE:
	cam_set_origin, cd, origin


 ARGUMENTS:
  INPUT:
	cd:	Array (nt) of CAMERA descriptors.

       origin:	Array (2,nt) of new origin values.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:  NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete

 	Written by:	Spitale, 1/1998
