NV_SET_NHIST


 PURPOSE:
	Changes the number of past states archived in a data descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	nv_set_nhist, dd, nhist


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor.

	nhist:	New nhist value.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
