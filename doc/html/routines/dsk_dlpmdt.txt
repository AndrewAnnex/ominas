DSK_DLPMDT


 PURPOSE:
	Returns dlpmdt for each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	dlpmdt = dsk_dlpmdt(dkx)


 ARGUMENTS:
  INPUT: NONE
	dkx:	 Any subclass of DISK.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 

  OUTPUT: NONE


 RETURN:
	dlpmdt value associated with each given disk descriptor.


 STATUS:
	Complete

 	Written by:	Spitale, 1/1998
