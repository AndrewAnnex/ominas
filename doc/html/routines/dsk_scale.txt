DSK_SCALE


 PURPOSE:
	Returns scale for each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	scale = dsk_scale(dkx)


 ARGUMENTS:
  INPUT: NONE
	dkx:	 Any subclass of DISK.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 

  OUTPUT: NONE


 RETURN:
	scale value associated with each given disk descriptor.


 STATUS:
	Complete
