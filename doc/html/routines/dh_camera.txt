DH_CAMERA


 PURPOSE:
	In conjunction with the detached header routine, write out
	camera parameters.


 CATEGORY:
       UTIL/SPICE


 CALLING SEQUENCE:
       dh_camera, dh, label, et, c_matrix


 ARGUMENTS:
  INPUT:
             dh:     Detached header name
	   label:     Vicar label from image
	      et:     Ephemeris time
	c_matrix:     C-matrix from SPICE


  OUTPUT:
       NONE

 KEYWORDS:
  INPUT:
       NONE

  OUTPUT:
       NONE



 STATUS:
       Completed.
