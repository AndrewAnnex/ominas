CLASS_GET


 PURPOSE:
	Returns the name of the object class.


 CATEGORY:
	NV/LIB/CLASS


 CALLING SEQUENCE:
	class = class_get(od)


 ARGUMENTS:
  INPUT:
	od:	 Descriptor of any class.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:  NONE

  OUTPUT: NONE


 RETURN:
	String giving the name of the object class.


 STATUS:
	Complete
