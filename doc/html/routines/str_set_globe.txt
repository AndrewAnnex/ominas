STR_SET_GLOBE


 PURPOSE:
	Replaces the globe descriptor in each given star descriptor.


 CATEGORY:
	NV/LIB/STR


 CALLING SEQUENCE:
	str_set_globe, sx, gbd


 ARGUMENTS:
  INPUT: 
	sx:	 Array (nt) of any subclass of STAR.

	gbd:	 Array (nt) of new globe descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete

 	Written by:	Spitale, 1/1998
