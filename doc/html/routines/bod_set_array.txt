BOD_SET_ARRAY


 PURPOSE:
	Replaces the array for the given body descriptor.


 CATEGORY:
	NV/LIB/BOD


 CALLING SEQUENCE:
	bod_set_array, bx, array


 ARGUMENTS:
  INPUT: 
	bx:	 BODY descriptor.

	array:	 Array of points in the body frame.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
