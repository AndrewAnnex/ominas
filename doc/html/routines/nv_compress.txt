NV_COMPRESS


 PURPOSE:
	Returns the compression function suffix associated with a data 
	descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	suffix = nv_compress(dd)


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: 
	String giving the compression suffix.  The full name
	of the compression function is nv_compress_data_<suffix>.


 STATUS:
	Complete
