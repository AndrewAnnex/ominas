SLD_SET_GM


 PURPOSE:
       Replaces the GM value for each given solid descriptor.


 CATEGORY:
	NV/LIB/SLD


 CALLING SEQUENCE:
	sld_set_gm, slx, gm


 ARGUMENTS:
  INPUT: 
	slx:	 Array (nt) of any subclass of SOLID descriptors.

	gm:	 Array (nt) of new GM values.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
