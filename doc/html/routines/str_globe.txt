STR_GLOBE


 PURPOSE:
       Returns a globe descriptor for each given star descriptor.


 CATEGORY:
       NV/LIB/STR


 CALLING SEQUENCE:
       gbd = str_globe(sd)


 ARGUMENTS:
  INPUT:
       sd:    Array (nt) of star descriptors

  OUTPUT:
       NONE


 KEYWORDS:
         NONE

 RETURN:
       An array (nt) of globe descriptors.


 STATUS:
       Completed.

       Written by:     Haemmerle, 5/1998
