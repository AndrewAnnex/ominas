CAT_VECTORS


 PURPOSE:
       To concatenate arrays of column vectors


 CATEGORY:
       UTIL


 CALLING SEQUENCE:
       result = cat_vectors(ps)


 ARGUMENTS:
  INPUT:
       ps:     An array of vector arrays.

  OUTPUT:
       NONE

 RETURN:
       Concatenated array of column vectors.


 STATUS:
       Completed.
