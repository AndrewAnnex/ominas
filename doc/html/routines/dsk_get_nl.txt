DSK_GET_NL


 PURPOSE:
	Obtains the nl (number of vertical harmonics) value from the enironment.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	nm = dsk_get_nm()


 ARGUMENTS:
  INPUT: NONE

  OUTPUT: NONE


 KEYWORDS:
  INPUT:  NONE

  OUTPUT: NONE


 RETURN:
	Value for nl obtained from the DSK_NL environmet variable.  Default
	is 4.


 STATUS:
	Complete

 	Written by:	Spitale
