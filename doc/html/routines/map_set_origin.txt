MAP_SET_ORIGIN


 PURPOSE:
	Replaces the origin for each given map descriptor.


 CATEGORY:
	NV/LIB/MAP


 CALLING SEQUENCE:
	map_set_origin, md, origin


 ARGUMENTS:
  INPUT: 
	md:	 Array (nt) of map descriptors.

	origin:	 Array (2,nt) of new map origins.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
