CAM_SET_FILTERS


 PURPOSE:
	Sets the filter name(s) for a given camera descriptor.


 CATEGORY:
	NV/LIB/CAM


 CALLING SEQUENCE:
	cam_set_filters, cd, filters, i


 ARGUMENTS:
  INPUT:
	cd:	 Camera descriptor.

	filters:	Name(s) of filters.

	index:	 Index of filter to set.  If not given, all are set.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:  NONE

  OUTPUT: NONE


 RETURN: NONE

 STATUS:
	Complete
