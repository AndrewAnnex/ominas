TAG_EXISTS


 PURPOSE:
	Determines whether a structure contains a given field.


 CATEGORY:
	UTIL


 CALLING SEQUENCE:
	result = tag_exists(struct, tag)


 ARGUMENTS:
  INPUT:
	struct:	Structure to test.

	tag:	String giving tag name to test for.

 RETURN:
	True if the tag exists, false otherwise.
