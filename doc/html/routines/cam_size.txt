CAM_SIZE


 PURPOSE:
	Returns the image size (in pixels) for each given camera descriptor.


 CATEGORY:
	NV/LIB/CAM


 CALLING SEQUENCE:
	size = cam_size(cd)


 ARGUMENTS:
  INPUT:
	cd:	 Array (nt) of CAMERA descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:  NONE

  OUTPUT: NONE


 RETURN:
	Size array associated with each given camera descriptor.


 STATUS:
	Complete
