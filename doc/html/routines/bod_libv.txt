BOD_LIBV


 PURPOSE:
	Returns the libration vector for each given body descriptor.


 CATEGORY:
	NV/LIB/BOD


 CALLING SEQUENCE:
	libv = bod_libv(bx)


 ARGUMENTS:
  INPUT: NONE
	bx:	 Array (nt) of any subclass of BODY descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 

  OUTPUT: NONE


 RETURN:
	Libration vector associated with each given body descriptor.


 STATUS:
	Complete
