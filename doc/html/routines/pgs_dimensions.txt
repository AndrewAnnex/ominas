PGS_DIMENSIONS


 PURPOSE:
	Determines dimensions of the points arrays on the given input 
	points arrays.


 CATEGORY:
	NV/PGS


 CALLING SEQUENCE:
	pgs_dimensions, ps, nv=nv, nt=nt


 ARGUMENTS:
  INPUT:
	ps:		Array of points structures.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: 
	nv:	Number of vectors/points.

	nt:	Number of timesteps/dimensions.


 RETURN: NONE
