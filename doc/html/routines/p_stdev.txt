P_STDEV


 PURPOSE:
       Computes the standard deviations of the given array of column vectors
	in the x direction.

 CATEGORY:
       UTIL/V


 CALLING SEQUENCE:
       result = p_stdev(v)


 ARGUMENTS:
  INPUT:
       v:     An array of np x nt points (i.e. 2 x np x nt).


  OUTPUT:
       NONE

 RETURN:
       Array of 1 x nt points (i.e., 2 x 1 x nt).


 STATUS:
       Completed.
