PG_POINTS


 PURPOSE:
	Concatenates the given pg_points_struct's into an array of image
	points.


 CATEGORY:
	NV/PG


 CALLING SEQUENCE:
	result = pg_points(object_ps)


 ARGUMENTS:
  INPUT:
	object_ps:	Array of pg_points_struct.	

 KEYWORDS:
  INPUT: 
	all:	If set, all points are returned, regardless of flag settings.

	name:	If set, points are returned only from points structures
		with the these names.

  OUTPUT: NONE


 RETURN:
	Array (2,n) of image points.


 STATUS:
	Complete


 SEE ALSO:
	pg_vectors
