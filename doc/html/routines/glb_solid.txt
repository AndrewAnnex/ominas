GLB_SOLID


 PURPOSE:
	Returns the solid descriptor for each given globe descriptor.


 CATEGORY:
	NV/LIB/GLB


 CALLING SEQUENCE:
	sld = glb_solid(gbx)


 ARGUMENTS:
  INPUT: 
	gbx:	 Array (nt) of any subclass of GLOBE descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	Solid descriptor associated with each given globe descriptor.


 STATUS:
	Complete
