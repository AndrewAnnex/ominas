DEGEN

 PURPOSE:
       Removes degenerate trailing dimensions.


 CATEGORY:
       UTIL


 CALLING SEQUENCE:
       result = degen(array)


 ARGUMENTS:
  INPUT:
       array:  An array.

  OUTPUT:
       NONE


 RETURN:
       Array, with any trailing dimensions of length 1 removed.


 SIDE EFFECTS:
	The input array is modified.


 STATUS:
       Completed.
