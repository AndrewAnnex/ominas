MAP_TYPE


 PURPOSE:
	Returns the type name for each given map descriptor.


 CATEGORY:
	NV/LIB/MAP


 CALLING SEQUENCE:
	type = map_type(md)


 ARGUMENTS:
  INPUT: NONE
	md:	 Array (nt) of map descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 

  OUTPUT: NONE


 RETURN:
	Array (nt) of type names associated with each given map descriptor.


 STATUS:
	Complete
