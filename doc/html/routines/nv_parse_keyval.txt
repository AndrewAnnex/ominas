NV_PARSE_KEYVAL


 PURPOSE:
	Parses a string containing a keyword=value pair.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	nv_parse_keyval, keyval, keyword, value


 ARGUMENTS:
  INPUT:
	keyval:		String of the form <keyword>=<value>.

  OUTPUT:
	keyword:	String giving the keyword.

	value:		String giving the value.


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete

 	Written by:	Spitale
