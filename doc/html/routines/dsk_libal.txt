DSK_LIBAL


 PURPOSE:
	Returns libal for each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	libal = dsk_libal(dkx)


 ARGUMENTS:
  INPUT: NONE
	dkx:	 Any subclass of DISK.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 

  OUTPUT: NONE


 RETURN:
	libal value associated with each given disk descriptor.


 STATUS:
	Complete
