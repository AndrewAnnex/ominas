CLIP


 PURPOSE:
	Return array with first value clipped off.


 CATEGORY:
	UTIL


 CALLING SEQUENCE:
	result = clip(array)


 ARGUMENTS:
  INPUT:
	array:	array to be operated on


 RETURN:
	If arr is a vector, returns it with the first element clipped off.
