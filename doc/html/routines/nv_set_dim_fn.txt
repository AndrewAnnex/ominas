NV_SET_DIM_FN


 PURPOSE:
	Replaces the dimension function associated with a data descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	nv_set_dim_fn, dd, dim_fn


 ARGUMENTS:
  INPUT:
	dd:		Data descriptor.

	dim_fn:	New sampling function.

  OUTPUT:
	dd:	Modified data descriptor.


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete


 SEE ALSO:
	nv_dim_fn
