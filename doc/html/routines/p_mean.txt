P_MEAN


 PURPOSE:
       Computes the mean values of the given array of points
	in the y direction.

 CATEGORY:
       UTIL/V


 CALLING SEQUENCE:
       result = p_mean(v)


 ARGUMENTS:
  INPUT:
       v:     An array of np x nt points (i.e. 2 x np x nt).


  OUTPUT:
       NONE

 RETURN:
       Array of 1 x nt points (i.e., 2 x 1 x nt).


 STATUS:
       Completed.


 MODIFICATION HISTORY:
       Written by:     Spitale
