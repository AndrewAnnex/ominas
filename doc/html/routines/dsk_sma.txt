DSK_SMA


 PURPOSE:
	Returns sma for each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	sma = dsk_sma(dkx)


 ARGUMENTS:
  INPUT: NONE
	dkx:	 Any subclass of DISK.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 

  OUTPUT: NONE


 RETURN:
	sma value associated with each given disk descriptor.


 STATUS:
	Complete
