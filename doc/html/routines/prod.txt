PROD


 PURPOSE:
       Computes the product of the elements of the given array.


 CATEGORY:
       UTIL


 CALLING SEQUENCE:
       result = prod(array, dim)


 ARGUMENTS:
  INPUT:
       array:  An input array

         dim:  The dimension over which to multiply.

  OUTPUT:
       NONE

 KEYWORDS:
       NONE


 RETURN:
       The product of all the elements in the array.


 STATUS:
       Completed.
