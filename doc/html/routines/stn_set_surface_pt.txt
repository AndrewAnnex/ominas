STN_SET_SURFACE_PT


 PURPOSE:
	Replaces the surface_pt vector in each given station descriptor.


 CATEGORY:
	NV/LIB/STN


 CALLING SEQUENCE:
	stn_set_surface_pt, std, surface_pt


 ARGUMENTS:
  INPUT: 
	std:	Array (nt) of STATION descriptors.

	surface_pt:	Array (1,3,nt) of surface_pt vectors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
