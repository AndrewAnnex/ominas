NV_NHIST


 PURPOSE:
	Returns the number of archived data states.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	nhist = nv_nhist(dd)


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: 
	Integer giving the number of past data states archived
	in the data descriptor.


 STATUS:
	Complete
