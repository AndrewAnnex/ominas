SLD_BODY


 PURPOSE:
	Returns the body descriptor for each given solid descriptor.


 CATEGORY:
	NV/LIB/SLD


 CALLING SEQUENCE:
	bd = sld_body(slx)


 ARGUMENTS:
  INPUT: 
	slx:	 Array (nt) of any subclass of SOLID descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	Body descriptor associated with each given solid descriptor.


 STATUS:
	Complete

 	Written by:	Spitale, 1/1998
