DSK_SET_DLPMDT


 PURPOSE:
	Replaces dlpmdt in each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	dsk_set_dlpmdt, bx, dlpmdt


 ARGUMENTS:
  INPUT: 
	dkx:	 Array (nt) of any subclass of DISK.

	dlpmdt:	 New dlpmdt value.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete

 	Written by:	Spitale, 1/1998
