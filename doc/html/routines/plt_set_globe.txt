PLT_SET_GLOBE


 PURPOSE:
	Replaces the globe descriptor in each given planet descriptor.


 CATEGORY:
	NV/LIB/PLT


 CALLING SEQUENCE:
	plt_set_globe, px, gbd


 ARGUMENTS:
  INPUT: 
	px:	 Array (nt) of any subclass of PLANET.

	gbd:	 Array (nt) of new globe descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete

 	Written by:	Spitale, 1/1998
