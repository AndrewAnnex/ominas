PTR_COPY_RECURSE


 PURPOSE:
	Copies data from the given pointer and from all pointers pointed to 
	by it.


 CATEGORY:
	UTIL


 CALLING SEQUENCE:
	result = ptr_copy_recurse(p)


 ARGUMENTS:
  INPUT:
	p:	Pointer to be copied.

  OUTPUT:
	NONE


 KEYWORDS:
  INPUT:
	NONE

  OUTPUT:
	NONE


 STATUS:
	xx
