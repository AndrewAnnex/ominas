NV_FILETYPE


 PURPOSE:
	Returns the filetype associated with a data descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	filetype = nv_filetype(dd)


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: 
	String giving the filetype.


 STATUS:
	Complete
