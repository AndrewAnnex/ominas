NV_TEST_DD


 PURPOSE:
	Determines whether the argument is a valid data descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	test = nv_test_dd(dd)


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor to test.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: 
	True if the argument is present, is a valid pointer, and
	points to a data descriptor.


 STATUS:
	Complete
