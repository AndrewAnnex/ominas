MAP_FN_DATA_P


 PURPOSE:
	Returns the function data pointer for each given map descriptor.


 CATEGORY:
	NV/LIB/MAP


 CALLING SEQUENCE:
	fn_data_p = map_fn_data_p(md)


 ARGUMENTS:
  INPUT: NONE
	md:	 Array (nt) of map descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 

  OUTPUT: NONE


 RETURN:
	Function data pointer associated with each given map descriptor.


 STATUS:
	Complete
