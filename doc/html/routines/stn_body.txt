STN_BODY


 PURPOSE:
	Returns the body descriptor for each given station descriptor.


 CATEGORY:
	NV/LIB/STN


 CALLING SEQUENCE:
	bd = stn_body(std)


 ARGUMENTS:
  INPUT: 
	std:	 Array (nt) of STATION descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	Body descriptor associated with each given station descriptor.


 STATUS:
	Complete
