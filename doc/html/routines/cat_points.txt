CAT_POINTS


 PURPOSE:
       To concatenate arrays of image points.


 CATEGORY:
       UTIL


 CALLING SEQUENCE:
       result = cat_points(ps)


 ARGUMENTS:
  INPUT:
       ps:     An array of image point arrays.

  OUTPUT:
       NONE

 RETURN:
       Concatenated array of image points.


 STATUS:
       Completed.
