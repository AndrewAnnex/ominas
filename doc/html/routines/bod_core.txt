BOD_CORE


 PURPOSE:
	Returns the core descriptor for each given body descriptor.


 CATEGORY:
	NV/LIB/BOD


 CALLING SEQUENCE:
	crd = bod_core(bx)


 ARGUMENTS:
  INPUT: 
	bx:	 Any subclass of BODY.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	Core descriptor associated with each given body descriptor.


 STATUS:
	Complete

 	Written by:	Spitale, 1/1998
