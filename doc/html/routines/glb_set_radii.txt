GLB_SET_RADII


 PURPOSE:
       Replaces the triaxial radii for each given globe descriptor.


 CATEGORY:
	NV/LIB/GLB


 CALLING SEQUENCE:
	glb_set_radii, gbx, radii


 ARGUMENTS:
  INPUT: 
	gbx:	 Array (nt) of any subclass of GLOBE descriptors.

	radii:	 Array (3,nt) of new triaxial radii.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete

 	Written by:	Spitale, 1/1998
