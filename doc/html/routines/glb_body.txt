GLB_BODY


 PURPOSE:
	Returns the body descriptor for each given globe descriptor.


 CATEGORY:
	NV/LIB/GLB


 CALLING SEQUENCE:
	bd = glb_body(gbx)


 ARGUMENTS:
  INPUT: 
	gbx:	 Array (nt) of any subclass of GLOBE descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	Body descriptor associated with each given globe descriptor.


 STATUS:
	Complete

 	Written by:	Spitale, 1/1998
