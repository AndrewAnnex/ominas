SLD_SET_REFL_FN


 PURPOSE:
       Replaces the reflection function for each given solid descriptor.


 CATEGORY:
	NV/LIB/SLD


 CALLING SEQUENCE:
	sld_set_refl_fn, slx, refl_fn


 ARGUMENTS:
  INPUT: 
	slx:	 Array (nt) of any subclass of SOLID descriptors.

	refl_fn: Array (nt) of new reflection functions.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
