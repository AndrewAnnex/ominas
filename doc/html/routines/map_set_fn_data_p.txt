MAP_SET_FN_DATA_P


 PURPOSE:
	Replaces the function data pointer for each given map descriptor.


 CATEGORY:
	NV/LIB/MAP


 CALLING SEQUENCE:
	map_set_fn_data_p, md, fn_data_p


 ARGUMENTS:
  INPUT: 
	md:	 Array (nt) of map descriptors.

	fn_data_p:	 Array (nt) of new function data pointers.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
