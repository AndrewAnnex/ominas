MAP_CORE


 PURPOSE:
	Returns the core descriptor for each given map descriptor.


 CATEGORY:
	NV/LIB/MAP


 CALLING SEQUENCE:
	crd = map_core(md)


 ARGUMENTS:
  INPUT: NONE
	md:	 Array (nt) of map descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 

  OUTPUT: NONE


 RETURN:
	Array (nt) of core descriptors associated with each given map 
	descriptor.


 STATUS:
	Complete
