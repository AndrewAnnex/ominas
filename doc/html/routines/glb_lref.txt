GLB_LREF


 PURPOSE:
       Returns the longitude system reference for each given globe
       descriptor.


 CATEGORY:
	NV/LIB/GLB


 CALLING SEQUENCE:
	lref = glb_lref(gbx)


 ARGUMENTS:
  INPUT:
	gbx:	 Array (nt) of any subclass of GLOBE descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:  NONE

  OUTPUT: NONE


 RETURN:
       Longitude system reference associated with each given globe
       descriptor.


 STATUS:
	Complete

 	Written by:	Spitale, 1/1998
