DSK_DLIBMDT


 PURPOSE:
	Returns dlibmdt for each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	dlibmdt = dsk_dlibmdt(dkx)


 ARGUMENTS:
  INPUT: NONE
	dkx:	 Any subclass of DISK.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 

  OUTPUT: NONE


 RETURN:
	dlibmdt value associated with each given disk descriptor.


 STATUS:
	Complete
