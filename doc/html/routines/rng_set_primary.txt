RNG_SET_PRIMARY


 PURPOSE:
	Replaces the primary string in each given ring descriptor.


 CATEGORY:
	NV/LIB/RNG


 CALLING SEQUENCE:
	rng_set_primary, rd, primary


 ARGUMENTS:
  INPUT: 
	rd:	Array (nt) of STATION descriptor.

	primary:	Array (nt) of primary strings.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
