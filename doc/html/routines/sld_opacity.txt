SLD_OPACITY


 PURPOSE:
	Returns the opacity for each given solid descriptor.


 CATEGORY:
	NV/LIB/SLD


 CALLING SEQUENCE:
	opacity = sld_opacity(slx)


 ARGUMENTS:
  INPUT: NONE
	slx:	 Any subclass of SOLID.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 

  OUTPUT: NONE


 RETURN:
	Opacity value associated with each given solid descriptor.


 STATUS:
	Complete
