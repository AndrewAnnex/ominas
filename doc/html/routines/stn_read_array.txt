STN_READ_ARRAY


 PURPOSE:
	Reads an station array file.


 CATEGORY:
	NV/LIB/BOD


 CALLING SEQUENCE:
	array = stn_read_array(filename)


 ARGUMENTS:
  INPUT: 
	filename:	 String giving the name of the file.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: Array of body-frame points.


 STATUS:
	Complete
