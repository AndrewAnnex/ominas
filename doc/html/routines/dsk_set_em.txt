DSK_SET_EM


 PURPOSE:
	Replaces em in each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	dsk_set_em, bx, em


 ARGUMENTS:
  INPUT: 
	dkx:	 Array (nt) of any subclass of DISK.

	em:	 New em value.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
