GET_IMAGE_VECTOR


 PURPOSE:
	Projects inertial vectors into an image.


 CATEGORY:
       NV/LIB/TOOLS


 CALLING SEQUENCE:
       p = get_image_vector(cd, v)


 ARGUMENTS:
  INPUT:
	cd:	Array (nt) of camera descriptors.

	v:	Array (nv,3,nt) of vectors in the inertial frame.

  OUTPUT:
       NONE


 KEYOWRDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: 
	Array (2,nv,nt) of image vectors.
