SLD_MASS


 PURPOSE:
       Returns the mass for each given solid descriptor.


 CATEGORY:
	NV/LIB/SLD


 CALLING SEQUENCE:
	mass = sld_mass(slx)


 ARGUMENTS:
  INPUT:
	slx:	 Array (nt) of any subclass of SOLID descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:  NONE

  OUTPUT: NONE


 RETURN:
       Array (nt) of mass values associated with each given solid 
	descriptor.


 STATUS:
	Complete
