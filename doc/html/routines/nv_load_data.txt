NV_LOAD_DATA


 PURPOSE:
	Loads the data array for a given data descriptor.  Adds to 
	NV state maintained list if maintain == 1.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	data = nv_load_data(dd)


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor to test.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: 
	Loaded data array.


 STATUS:
	Complete
