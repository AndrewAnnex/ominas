STN_SURFACE_PT


 PURPOSE:
	Returns the surface_pt vector for each given station descriptor.


 CATEGORY:
	NV/LIB/STN


 CALLING SEQUENCE:
	surface_pt = stn_surface_pt(std)


 ARGUMENTS:
  INPUT: 
	std:	 Array (nt) of STATION descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	Array (1,3,nt) of surface_pt vectors associated with each given 
	station descriptor.


 STATUS:
	Complete
