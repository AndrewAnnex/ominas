COR_NAME


 PURPOSE:
	Returns the name for each given core descriptor.


 CATEGORY:
	NV/LIB/COR


 CALLING SEQUENCE:
	name = cor_name(crx)


 ARGUMENTS:
  INPUT: NONE
	crx:	 Any subclass of CORE.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 

  OUTPUT: NONE


 RETURN:
	Name associated with each given core descriptor.


 STATUS:
	Complete
