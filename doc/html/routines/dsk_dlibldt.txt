DSK_DLIBLDT


 PURPOSE:
	Returns dlibldt for each given disk descriptor.


 CATEGORY:
	NV/LIB/DSK


 CALLING SEQUENCE:
	dlibldt = dsk_dlibldt(dkx)


 ARGUMENTS:
  INPUT: NONE
	dkx:	 Any subclass of DISK.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: 

  OUTPUT: NONE


 RETURN:
	dlibldt value associated with each given disk descriptor.


 STATUS:
	Complete
