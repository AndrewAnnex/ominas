NWHERE


 PURPOSE:
       Finds subscripts where an element in the first array matches
       elements in the second input array.


 CATEGORY:
       UTIL


 CALLING SEQUENCE:
       result = nwhere(ref, list)


 ARGUMENTS:
  INPUT:
        ref:  Reference array

       list:  Comparison array.


  OUTPUT:
       NONE

 KEYWORDS:
       NONE


 RETURN:
       An array of subscripts in ref that match something in list.


 STATUS:
       Completed.
