RNG_DISK


 PURPOSE:
	Returns the disk descriptor for each given ring descriptor.


 CATEGORY:
	NV/LIB/RNG


 CALLING SEQUENCE:
	dkd = rng_disk(rd)


 ARGUMENTS:
  INPUT: 
	rd:	 Array (nt) of RING descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	Disk descriptor associated with each given ring descriptor.


 STATUS:
	Complete
