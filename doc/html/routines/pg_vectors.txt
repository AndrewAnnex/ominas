PG_VECTORS


 PURPOSE:
	Concatenates the given pg_points_struct's into an array of inertial
	vectors.


 CATEGORY:
	NV/PG


 CALLING SEQUENCE:
	result = pg_vectors(object_ps)


 ARGUMENTS:
  INPUT:
	object_ps:	Array of pg_points_struct.	



 RETURN:
	Array (2,n) of image points.


 STATUS:
	Complete


 SEE ALSO:
	pg_points
