NV_DIM_FN


 PURPOSE:
	Returns the dimension function associated with a data descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	dim_fn = nv_dim_fn(dd)


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	The dim_fn associated with the data descriptor.


 STATUS:
	Complete


 SEE ALSO:
	nv_set_dim_fn
