MAP_SET_GRAPHIC


 PURPOSE:
	Replaces the graphic flag for each given map descriptor.


 CATEGORY:
	NV/LIB/MAP


 CALLING SEQUENCE:
	map_set_graphic, md, graphic


 ARGUMENTS:
  INPUT: 
	md:	 Array (nt) of map descriptors.

	graphic:	 Array (nt) of new graphic flags.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete
