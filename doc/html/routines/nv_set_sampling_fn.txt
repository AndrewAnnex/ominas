NV_SET_SAMPLING_FN


 PURPOSE:
	Replaces the sampling function associated with a data descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	nv_set_sampling_fn, dd, sampling_fn


 ARGUMENTS:
  INPUT:
	dd:		Data descriptor.

	sampling_fn:	New sampling function.

  OUTPUT:
	dd:	Modified data descriptor.


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete


 SEE ALSO:
	nv_sampling_fn
