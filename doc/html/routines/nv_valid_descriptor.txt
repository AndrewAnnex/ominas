NV_VALID_DESCRIPTOR


 PURPOSE:
	Determines whether the argument is a valid data descriptor,
	or data descriptor structure.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	test = nv_valid_descriptor(dd)


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor to test.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN: 
	True if the argument is a data descriptr structure or a
	pointer to one.


 STATUS:
	Complete
