NV_INSTRUMENT


 PURPOSE:
	Returns the instrument string associated with a data descriptor.


 CATEGORY:
	NV/SYS


 CALLING SEQUENCE:
	instrument = nv_instrument(dd)


 ARGUMENTS:
  INPUT:
	dd:	Data descriptor.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	The instrument string associated with the data descriptor.


 STATUS:
	Complete
