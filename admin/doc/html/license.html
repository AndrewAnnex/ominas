MINAS License Generation

Use the following procedure to produce a functioning MINAS license file from a prototype license file (see MINAS installation):

1) Change directory to the MINAS administarator directory $MINAS/admin.

2) Copy the prototpe file to this directory.

3) Start idl.

4) Enter the command:

    lic_gen
A license file will be written to the current directory. Its name is specified by the MINAS_LICENSE environment variable.
